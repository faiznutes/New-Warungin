## ðŸŽ“ KEY CONCEPTS YOU NEED TO KNOW

### Consolidation
Taking 3-5 pages that do similar things and merging them into 1 smart page with tabs/sections/modals.

**Example**: Orders + Kitchen Orders + Failed Syncs â†’ 1 Orders Management page

### Role-Based Conditional Rendering
Same component renders differently based on user role (ADMIN/SUPERVISOR/CASHIER/KITCHEN).

**Example**: Single Dashboard.vue with conditional content per role

### Addon-Gated Routes
Routes only available if tenant has that addon enabled.

**Example**: Marketing routes require DELIVERY_MARKETING addon

### Lazy Loading
Components loaded only when needed, not all at once.

**Example**: `component: () => import('@/views/operational/OrdersManagement.vue')`
